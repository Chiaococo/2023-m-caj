# Installation
OpenShift Pipelines 作為 OpenShift 之上的附加元件提供，可透過 OpenShift OperatorHub。請參考 [these instructions](https://github.com/openshift/pipelines-tutorial/blob/master/install-operator.md) 以便透過 OperatorHub 在 OpenShift 上安裝 OpenShift Pipelines。
